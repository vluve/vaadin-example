# Vaadin Demo App

Демо-приложение на **Vaadin 24 + Spring Boot 3**, показывающее:

* Кладовую с ингредиентами
* Список рецептов с изображениями, рейтингом и доп. информацией
* Страница с наверстанным резюме

## Технологии

* Java 21
* Spring Boot 3.5
* Vaadin 24
* Spring JPA + PostgreSQL
* Maven

## Структура проекта

1. Интерфейс лежит в src/main/java/com/vluve/examplefeature/ui
2. Классы для хранения и обработки данных лежат в src/main/java/com/vluve/demoapp
